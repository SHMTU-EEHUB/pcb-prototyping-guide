# 设计建议

```admonish warning title="这里与常规制板明显不同，请特别注意"
下面三条**不是**通用 PCB 最佳实践，而是**针对"无电镀"工艺的专门调整**,与常规（有电镀的）制板做法差异显著。

按平时设计 PCB 的习惯来布线，虽然能够正常使用，但可能在焊接过程中遇到一些"困难"——所以尽管名字叫"建议"，也**尽量遵守**。
```

## 建议 1：过孔宁大勿多 {#tip-1}

不要为了换层连续堆一堆小过孔；用**少量大过孔**替代。

```admonish note title="为什么？" collapsible=true
每个过孔都要人手穿铁丝 + 两面焊——孔越多，工作量爆炸式增长。

一个大过孔（比如 1 mm 以上）有几个好处：

- 铁丝更容易穿过
- 焊锡更容易吃进去、焊点更牢
- 一个大孔能顶多个 0.3 mm 小孔的连接能力
```

## 建议 2：换层过孔放在直插件**旁边** {#tip-2}

当直插件的某个引脚需要换层连接时，**不要**想着"在直插件自己的孔上想办法"——在元件**边上**放一个单独的过孔，让顶层走线先到这个过孔再换层到底层。

```admonish note title="为什么？" collapsible=true
- 直插件自己的孔被元件引脚占据，穿不进铁丝
- 有些直插件（IC 底面、排母等）引脚两面都要焊是做不到的——被元件本体挡住
- 在旁边单独放一个过孔没有这些限制，还能用上[建议 1](#tip-1)（直接做成大过孔）
```

## 建议 3：背面走信号，正面用作地平面 {#tip-3}

信号走线尽量全部放在**底层（焊接面）**,把**顶层**整片敷铜作为 GND 地平面。

```admonish note title="为什么？" collapsible=true
- **焊接更容易**:走线和元件引脚在同一面，焊接时连接直观，不用为了连一根线专门加换层过孔
- **顶层走信号容易短路**:如果把走线留在顶层（元件面），焊接时焊锡很容易从焊盘流到相邻的走线上造成短路
```
