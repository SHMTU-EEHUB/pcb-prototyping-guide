# 设计规则

这两条是**硬性约束**,由刀具和工艺决定，必须遵守。

## 规则 1：最小间距和走线宽度都不小于 0.5 mm {#rule-1}

| 参数         | 数值                 | 说明                            |
| ------------ | -------------------- | ------------------------------- |
| 官方最小间距 | 0.1 mm               | 官方指标[^dm350-spec]           |
| 实际下限     | 0.2 mm               | 受限于我们使用的刀具直径 0.2 mm |
| **建议值**   | **0.5 mm (≈20 mil)** | 线宽和间距都按这个设计          |

```admonish note title="为什么不按官方的 0.1 mm？" collapsible=true
- **刀具直径 0.2 mm**:物理下限就是 0.2 mm，官方标的 0.1 mm 我们做不出来
- **切割容易不干净**:小间距雕刻时常有铜屑残留，可能造成意外短路
- **无绿油容易连锡**:我们的板子不上阻焊层，焊盘之间没有绿油隔离，间距太小焊接时很容易连成一片
```

## 规则 2：过孔直径从固定列表里选 {#rule-2}

我们的钻头种类有限，能用的孔径只有以下几种（单位 mm）:

**0.6、0.7、0.8、0.9、1.0、1.1、1.2、1.3、1.5、3.0**

不在这个列表里的孔径**会被自动映射到最接近的值**。

```admonish warning title="不建议使用 0.8 mm 以下的孔"
0.8 mm 以下的孔**穿不进常见的铁丝**。换层依赖铁丝桥接，孔径过小等于该过孔实际不导通——对照[**建议 1**](./tips.md#tip-1),用大过孔通常也更合适。
```

[^dm350-spec]: [DM350 官方文档](../docs/快速电路板制作.pdf#page=16)
