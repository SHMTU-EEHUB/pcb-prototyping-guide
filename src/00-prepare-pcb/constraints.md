# 核心约束

## 1. 所有通孔默认不导通

**做法**:焊接时用一段细铁丝穿过孔，正反两面各焊一次。

```admonish note title="原因" collapsible=true
"通孔两面连通"靠的是孔壁的镀铜层——正好是我们做不了的那一步。没有孔壁铜，上下两层必须靠别的东西桥接；最实用的就是插一根细铁丝当桥，再用焊锡固定。
```

## 2. 直插件的焊盘两面不连通

直插元件（排针、插件电阻电容、DIP 芯片等）的焊盘天然跨正反两面，但两面**默认不电气连接**。即使引脚插进去，顶层铜环和底层铜环之间仍然断开。

```admonish note title="原因" collapsible=true
直插件的焊盘由一个贯穿板子的孔加周围的铜环构成。正常 PCB 这个孔是电镀的，所以两面焊盘自动连通；我们的板子不电镀，两面就是两个独立的铜环。
```

## 3. DRC 完全无法使用

```admonish warning title="不要依赖 DRC"
EDA 软件的设计规则检查默认所有通孔都是导通的。在我们的限制下这个假设**完全不成立**:

- DRC 看起来"全过"，但实际电路可能到处都是断路
- 布线完成后**必须手动**逐个检查每一个直插件、每一个引脚的走线情况
```
