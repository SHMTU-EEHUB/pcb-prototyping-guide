# 4. 启动 DM350 机床

**等 DM350 电脑启动完成后**,再执行以下三步把机床本体也通上电：

## 4.1 关闭急停

**逆时针**旋转急停按钮，让它弹起——急停解除。

![急停按钮](../images/dm350_急停按钮.jpg)

## 4.2 转动钥匙开关

**顺时针**转动钥匙开关。

![钥匙开关](../images/dm350_钥匙开关.jpg)

## 4.3 按上电按钮

按下**上电按钮**,绿色指示灯点亮，机床通电完成。

![上电指示灯点亮](../images/dm350_上电按钮.jpg)

上电完成后整机样式：

![上电完成](../images/dm350_开机完成.jpg)
