# 简介

## 本教程是什么

这是我们社团**制板机（DM350）** 的使用指南，涵盖从 PCB 设计准备、Gerber 导出、刀路计算、正式加工到最后关机的完整流程。

## 适用对象

所有希望使用制板机打板的社团成员。

```admonish warning title="先泼盆冷水"
制板机**没你想象的那么好用**。它的限制相当多（详见[第 0 步](./00-prepare-pcb.md)），单板制作耗时常以"半天"为单位，后续焊接也比常规 PCB 费劲。

**只要条件允许，优先使用成熟的商业打板服务（嘉立创、华秋 等）**。真正合适用制板机的场景其实只有几种：

- 当天就要拿到板子（如电赛期间）
- 样品改动极频繁，等不起商业打板的物流
- 学习、体验制板流程本身
```

## 设备信息（DM350）

社团使用的是 **DM350** 型快速电路板制作机。下表列出官方技术参数[^dm350-spec]:

| 项目                 | 值                     |
| -------------------- | ---------------------- |
| 最小孔径             | 0.15 mm                |
| 最小线宽             | 0.075 mm               |
| 最小线间距           | 0.1 mm                 |
| 最大加工幅面         | 305 × 230 mm           |
| 重复定位精度         | ≤ ±0.002 mm            |
| 系统定位精度         | ≤ ±0.01 mm             |
| 分辨率               | 0.5 μm                 |
| 激光波长             | 无（DM350 无激光模块） |
| 主轴转速             | 80,000 rpm             |
| 刀具库               | 20 把刀位              |
| 换刀方式             | 自动换刀               |
| 刀深调节             | 自动调整               |
| 限位方式             | 接触式限位             |
| 钻孔能力             | 150 次/min             |
| 空载移动速度         | 250 mm/s               |
| 电源                 | 220 VAC / 50 Hz        |
| 功率                 | 1.6 kW                 |
| 设备尺寸 (W × H × D) | 880 × 900 × 1,450 mm   |
| 设备重量             | 200 kg                 |

实际可用约束（如刀具直径、建议间距）通常**严于**表中的官方最小值，详情见[第 0 步](./00-prepare-pcb.md)。

[^dm350-spec]: [DM350 官方文档](./docs/快速电路板制作.pdf#page=16)

## 整体流程

本教程按章节编号排列内容，但**实际操作顺序**并不是从第 0 步顺序走到第 6 步——设备预热（第 2 步）要约 25 分钟，通常在预热的同时进行 Gerber 导出（第 1 步）节省等待。

```mermaid
flowchart LR
    S0["第 0 步<br/>准备 PCB"] --> S2["第 2 步<br/>开机 + 预热<br/>(≈25 分钟)"]
    S2 --> S1["第 1 步<br/>导出 Gerber"]
    S1 --> S3["第 3 步<br/>刀路计算"]
    S3 --> S4["第 4 步<br/>制板准备"]
    S4 --> S5["第 5 步<br/>正式制板"]
    S5 --> S6["第 6 步<br/>关机"]
```

```admonish tip title="时间利用建议"
第 2 步开始预热后，机器自行加热约 25 分钟。这段时间请同步进行第 1 步（导出 Gerber），充分利用等待时间。
```
