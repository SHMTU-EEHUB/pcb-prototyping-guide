# 3. 铜板定位

## 3.1 拆除垫板

把刚才钉好的垫板拆下来，基座恢复到裸露状态：

![拆除垫板后的基座](../images/定位_拆除垫板.jpg)

## 3.2 取销钉

在**窗台**上找到销钉：

![销钉](../images/定位_销钉.jpg)

## 3.3 把销钉装到基座

**倒角的一侧朝下**,把销钉插进基座对应位置的孔里：

![销钉安装完成](../images/定位_销钉已装.jpg)

## 3.4 依次盖上纸板和铜板

先把纸板**对准销钉孔**盖上：

![纸板已盖](../images/定位_纸板已盖.jpg)

再把铜板对准销钉孔盖上——**铜板就被销钉卡住定位**了：

![铜板已盖](../images/定位_铜板已盖.jpg)
