# 3. 关闭气泵与断开气管

## 3.1 关闭气泵

主机电源关闭后，**按气泵面板开关关闭气泵**。

## 3.2 断开气管

1. 把气泵阀门**顺时针拧到 OFF**(阀门横着)
2. 按住阀门接口上的**蓝色锁紧按钮**,拔出气管

![气泵阀门关闭 + 气管已拔](../images/关机_气泵阀门拔管.jpg)

```admonish danger title="一定要先关阀门再拔管"
阀门处于 ON 时管内仍有高压气体，**直接拔管会让气管弹飞**——可能打到人或周围东西。顺序必须是**先关阀门，再拔管**。
```

## 3.3 排气 + 离开

管子拔掉后，阀门另一侧（气泵里）还存着压缩空气。**缓缓打开阀门**,把剩余气放光即可。

```admonish tip title="也可以快一点"
也可以**一下子拧开**,然后**快速逃离教室**——瞬间排气的声音相当响亮。
```

最后确认所有东西（垫板、销钉、纸板、成品、工具）都**放回原位**,关灯离开。
