# 第 0 步：准备 PCB

```admonish abstract title="TL;DR"
- 设备不能电镀 → 所有通孔**正反两面默认不连通**
- 解决办法：焊接时用**铁丝穿孔 + 两面各焊一次**
- 直插件的焊盘也一样不连通，设计时要当心
- **不要依赖 DRC**
- 硬规则：
  - 最小间距和线宽 **≥ 0.5 mm**
  - 过孔直径**只能从固定列表选**,且不要低于 0.8 mm
- 设计建议：
  - 过孔宁**大勿多**：用少量大过孔替代一堆小过孔
  - 过孔**放在直插件旁边**,不要指望直插件自己的孔换层
  - **背面走信号，正面作地平面**
```
